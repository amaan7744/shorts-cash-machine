# Shorts Cash Machine

Automated YouTube Shorts system designed for short-term monetization.

## What this does
- Finds already viral YouTube videos (2M+ views)
- Extracts a short clip
- Generates AI commentary
- Generates AI voiceover
- Creates a Shorts-ready video
- Uploads automatically with creator credit

⚠️ This is a short-term cash system. Not intended for long-term brand building.

---

## Requirements
- Python 3.10+
- YouTube Data API v3
- YouTube upload credentials
- OpenAI API key

---

## Run locally
```bash
pip install -r requirements.txt
python workflow/run.py
